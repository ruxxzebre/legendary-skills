# legendary-skills

Skills for software development and day-to-day activities (for AI and agents).

A collection of reusable [agent skills](https://agentskills.io) that extend AI coding agents with structured workflows and domain knowledge.

## Skills

| Skill | Description |
|-------|-------------|
| **changelog** | Write structured changelog entries following [Keep a Changelog](https://keepachangelog.com/) format. Supports chat, git, and hybrid modes. |

## Installation

Install using the [skills CLI](https://github.com/vercel-labs/skills):

```bash
# Install all skills
npx skills add ruxxzebre/legendary-skills

# Install a specific skill
npx skills add ruxxzebre/legendary-skills --skill changelog

# Install to a specific agent
npx skills add ruxxzebre/legendary-skills -a claude-code
```
